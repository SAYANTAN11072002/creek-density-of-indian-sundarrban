CREEK DENSITY OF INDIAN SUNDARBAN

📄 ABSTRACT
Creek density is a vital geomorphological indicator in the Sundarban delta, influencing tidal dynamics, sediment transport, and mangrove health. This study analyses multi‑temporal creek networks using satellite imagery (1991–2022) to quantify changes in drainage density and sinuosity. The interactive maps below, generated via QGIS2Web, allow visual comparison of creek density across different decades. Each viewer has a YEAR button to load the corresponding creek density map.
